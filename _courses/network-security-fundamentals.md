## Course Overview

Network Security Fundamentals provides a comprehensive introduction to securing computer networks. You'll learn both defensive techniques and common attack vectors to become a well-rounded security professional.

### What You'll Learn

- Understand network security principles and architectures
- Configure and manage firewalls
- Implement intrusion detection and prevention systems
- Design secure network topologies
- Deploy VPNs for secure communications
- Monitor networks for security threats

### Hands-On Labs

- Configure firewall rules in pfSense
- Set up Snort IDS
- Deploy an IPSec VPN
- Network traffic analysis with Wireshark
- Build a secure network architecture
