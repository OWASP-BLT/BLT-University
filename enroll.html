How to Enroll

BLT University uses GitHub for course enrollment and management. Follow these steps:

1

Choose Your Course

Browse our course catalog and select the course you want to enroll in.

2

Create GitHub Account

If you don't have one already, create a free GitHub account.

3

Submit Enrollment Request

Click the button below to open an enrollment issue on GitHub. Fill in the course details and your information.

4

Start Learning

Once enrolled, you'll receive access to course materials and be added to the course repository.
