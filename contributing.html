How to Create a Course

1

Fork the Repository

Start by forking the BLT-University repository on GitHub.

2

Create Instructor Profile

Add a new file in the _instructors directory:

---
layout: instructor
title: Your Name
expertise: Your Area of Expertise
---
Your bio goes here...
3

Create Your Course

Add a new file in the _courses directory:

---
layout: course
title: Course Title
category: Category
level: Beginner
---
## Module 1: Intro
Course content here...
4

Submit Pull Request

Submit a pull request. Once approved, your course merges and automatically goes live!
